# AI Engineer: Zero To Hero
## Project Overview: 

This repository is **comprehensive learning roadmap for aspiring AI Engineers**. It takes you from **Python Fundamentals -> AI Concepts -> Machine Learning -> Deep Learning -> Real World Projects.**

---
## Learning Path/Contents
1. Python Basics
   * Variables, datatypes, operators
   * Functions, classes, input/output
   * Simple Projects (Calculator, mini-games)

### Python Coding Problems: Zero to Hero for AI Journey

A comprehensive progression of coding challenges to master Python for artificial intelligence and machine learning.

---

#### Phase 1: Python Fundamentals (Weeks 1-2)

##### Level 1: Basic Syntax & Data Types
1. **Hello AI World** - Print a formatted message with variables
2. **Temperature Converter** - Convert Celsius to Fahrenheit and Kelvin
3. **BMI Calculator** - Calculate and categorize body mass index
4. **Even or Odd Classifier** - Determine if numbers are even or odd
5. **Leap Year Checker** - Identify leap years using conditional logic
6. **Grade Calculator** - Convert numerical scores to letter grades
7. **Simple Interest Calculator** - Calculate compound vs simple interest
8. **String Reverser** - Reverse strings without using built-in functions
9. **Palindrome Checker** - Identify palindromic words
10. **Vowel Counter** - Count vowels in a given string

##### Level 2: Data Structures
11. **List Statistics** - Calculate mean, median, mode from a list
12. **Remove Duplicates** - Create unique list from duplicates
13. **List Intersection** - Find common elements between lists
14. **Dictionary Merger** - Combine multiple dictionaries intelligently
15. **Frequency Counter** - Count word/character frequencies using dict
16. **Matrix Addition** - Add two 2D matrices (nested lists)
17. **Transpose Matrix** - Transpose a 2D matrix
18. **Set Operations** - Implement union, intersection, difference
19. **Nested Data Extractor** - Extract values from nested dictionaries
20. **Shopping Cart** - Build a cart system with items and prices
